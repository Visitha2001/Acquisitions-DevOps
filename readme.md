# ⚙️ Setup

```bash
npm i eslint @eslint/js prettier eslint-config-prettier eslint-plugin-prettier -D
```

## drizzle installation
```bash
npm i @neondatabase/serverless drizzle-orm
```

## drizzle kit installation
```bash
npm i -D drizzle-kit
```

## winston installation
```bash
npm i winston
```
